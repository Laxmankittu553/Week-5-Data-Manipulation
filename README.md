# Customer Sales Analysis

## Project Overview
This project analyzes customer purchasing behavior using Python and data analysis libraries. The objective is to identify top customers, analyze product sales performance, and observe monthly sales trends using sales data and customer information.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Description
The project uses two datasets:

1. **sales_data.csv**
   - Contains transaction-level sales information such as product, quantity, price, region, and total sales.

2. **customer_churn.csv**
   - Contains customer-related information such as tenure, monthly charges, contract type, and churn status.

## Features of the Analysis
- Data loading and exploration
- Revenue analysis
- Top customer identification
- Monthly sales trend analysis
- Product sales comparison
- Pivot table analysis
- Data visualization using charts

## Visualizations
The project includes several visualizations such as:
- Bar chart for top customers
- Line chart for monthly sales trend
- Bar chart for product sales
- Heatmap for regional sales analysis
